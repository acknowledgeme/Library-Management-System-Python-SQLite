# Library-Management-System-Python-SQLite
A python CRUD application using SQLite to manage book records
This project is a simple Library Management System developed using Python and SQLite.  
It demonstrates basic CRUD (Create, Read, Update, Delete) operations.

## Features
- Add new books
- View all books
- Update book details
- Delete books by ID

## Technologies Used
- Python
- SQLite
- SQL
